==================================================================
Getting and Cleaning Data Course Project
==================================================================

To run the run_analysis you need to follow the following steps:

Step 1. 
Download the Human Activity Recognition Using Smartphones Data Set
from this address: https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 

Step 2.
Unzip the download file below the project in the directory 'UCI HAR Dataset'.
You should have 4 files (activity_label.txt, features.text. features_info.tx and README.txt) in the directory 'UCI HAR Dataset' and 2 directories (test and train)

Step 3.
Make sure you set the directory in which the file run_analysis.R is in as the working the directory.

Step 4.
You can run the analysis by typing run() after you load the run_analysis.R code
